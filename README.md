# Interactive Prototype Creation

## Project Overview

This project demonstrates the creation of an interactive mobile app prototype for a College Event Management Application. The prototype simulates real user interactions, navigation flows, and screen transitions to provide a realistic user experience before development.

---

## Objective

To convert static wireframes into a clickable prototype and evaluate usability through user testing and feedback collection.

---

## Features

- Clickable navigation between screens
- Interactive user flow simulation
- Screen transitions and animations
- User testing and feedback analysis
- Design refinements based on testing results

---

## User Flow

Login → Home → Event Details → Registration → Success Screen

---

## Prototype Screens

### 1. Login Screen
- Email input
- Password input
- Login button

### 2. Home Screen
- Event search
- Featured events
- Event categories

### 3. Event Details Screen
- Event information
- Date and venue details
- Register button

### 4. Registration Success Screen
- Confirmation message
- Navigation to My Events

---

## Interactions Implemented

- Button click navigation
